The GitHub API issues path is: https://api.github.com/repos/[username]/[project]/issues

Initalizing the Class ( Available Fields )

  • debugger ( optional ) [boolean] - If true outputs program status to the console
  • domain ( optional ) [string] - The URL to the GitHub API. Should it change in the future
  • username ( required ) [string] - Your username as it corresponds to the repository you are targetting
  • title ( optional ) [string] - Title of the Issue to be displayed

Calling the API 

        /**
         * @method getIssues
         * @param project {String} : Name of the project to be displayed
         * @param target {String} : DOM element to be targetted
         *                          Use CSS selector standards
         */
        hasIssues.getIssues(project,target);
